## Загрузка через браузер (5 мин)

1. Откройте: https://github.com/Diabloalex666/Calculators  
2. **Add file → Upload files**  
3. Перетащите **всё** из `F:\AI\side-income`  
   (или распакуйте `Calculators-upload.zip` из этой же папки)  
4. **Commit changes**

Обязательно должны попасть в корень:

- `index.html`, все `.html`
- папки `css/`, `js/`, `.github/`
- `.nojekyll` (важно для GitHub Pages!)
- `robots.txt`, `sitemap.xml`

---

## Включить GitHub Pages

1. https://github.com/Diabloalex666/Calculators/settings/pages  
2. **Build and deployment → Source:** Deploy from a branch  
3. Branch: **main** → **/ (root)** → Save  

Через 1–5 минут откройте:

https://diabloalex666.github.io/Calculators/

---

## Если 404 не пропадает

- [ ] Файл `index.html` в **корне** репозитория (не в подпапке)
- [ ] В корне есть `.nojekyll`
- [ ] Settings → Pages — зелёный статус « deployed »
- [ ] Подождите до 10 минут после первого commit
