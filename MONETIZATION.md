# Как монетизировать FinPulse (один раз настроил — работает)

## Шаг 1. Публикация (бесплатно)

### Вариант A — GitHub Pages (проще всего)

1. Создайте репозиторий на GitHub, залейте папку `side-income`.
2. Settings → Pages → Source: branch `main`, folder `/ (root)`.
3. Сайт будет на `https://USERNAME.github.io/REPO/`.

### Вариант B — Cloudflare Pages

1. cloudflare.com → Pages → Create project → Upload assets.
2. Загрузите содержимое `side-income`.
3. Получите адрес `https://something.pages.dev` или привяжите свой домен.

### Вариант C — свой домен

- Купить `.ru` (REG.RU, Timeweb и т.д.).
- Направить DNS на GitHub Pages / Cloudflare.
- В `sitemap.xml`, `robots.txt`, `index.html` заменить `YOUR-DOMAIN.ru`.

## Шаг 2. Яндекс.Метрика (бесплатно)

1. metrika.yandex.ru → добавить счётчик.
2. Скопировать номер ID в `js/config.js` → `yandexMetrikaId`.
3. Добавить перед `</body>` на всех страницах (или один раз в общий скрипт):

## Шаг 3. Реклама Яндекс (РСЯ) — основной «пассив»

1. partner.yandex.ru → Рекламная сеть (РСЯ).
2. Добавить сайт, дождаться модерации (обычно 1–7 дней).
   - Нужен нормальный текст на страницах (у вас уже есть).
   - Не кликайте сами по своей рекламе — блокировка.
3. Создать адаптивный блок, вставить код в `index.html` и страницы калькуляторов под результатом расчёта.

**Ориентир дохода:** при 1000 визитов/мес и RPM 30–80 ₽ → **30–80 ₽ за 1000 показов**, итого **300–2400 ₽/мес** на старте. С ростом SEO растёт линейно.

## Шаг 4. Чаевые на Boosty (без продаж)

1. boosty.to → создать страницу (можно бесплатную подписку «поддержка»).
2. Ссылку вставить в `js/config.js` → `boostyUrl`.
3. На главной уже есть блок «Поддержать проект».

Даже 5–10 человек × 100 ₽ = приятный бонус без рекламы.

---

## Шаг 5. Цифровой продукт (опционально, 1 вечер)

Расширенный **«Бюджет на год»** продать на Boosty за **99–199 ₽**:

1. Дублируйте `budget.html` → добавьте 12 вкладок или таблицу на год.
2. «File → Print → Save as PDF».
3. На Boosty: пост «Шаблон бюджета PDF» + авто-выдача файла подписчикам.

Продажи идут из тех же калькуляторов: «Скачать расширенный шаблон».

---

## Шаг 6. SEO без «продаж» (15 мин/нед)

Раз в 1–2 недели добавляйте короткую статью-абзац на страницу:

- «Как посчитать отпускные при неполном месяце»
- «13 или 15 процентов НДФЛ в 2026»

Это бесплатный трафик из поиска. Не нужно звонить клиентам.

Полезные запросы:

- калькулятор зарплаты на руки
- калькулятор отпускных онлайн
- сложный процент калькулятор накоплений

---

## Шаг 7. Забыть и проверять раз в месяц

Раз в месяц (10 минут):

- [ ] Метрика: сколько визитов
- [ ] РСЯ: сколько начислено
- [ ] Boosty: были ли донаты

Если визитов < 100 — добавьте 1 SEO-абзац или ссылку в профиль/форум (не спам, а «вот полезный калькулятор»).

## Почему это подходит под «есть работа, хобби»

| Критерий | FinPulse |
|----------|----------|
| Время на поддержку | ~10 мин/мес |
| Продажи | Не нужны |
| Сервер | Не нужен |
| Юридическое | Достаточно ИП/самозанятость когда доход > 0 регулярно (уточните у бухгалтера) |

Когда пойдут стабильные выплаты с РСЯ — оформите **самозанятость** и принимайте чеки через «Мой налог» (занимает 15 минут).
