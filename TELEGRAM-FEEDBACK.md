# Обратная связь → Telegram (без показа вашего ника)

Пользователи видят только форму на сайте. Сообщения приходят **вам в личный Telegram** через бота. **Ваш ник на сайте нигде не показывается.**

Настройка один раз, ~15 минут.

---

## Шаг 1. Создайте бота (5 мин)

1. Откройте Telegram → **@BotFather**
2. Команда `/newbot`
3. Имя бота: `FinPulse Feedback` (любое)
4. Username бота: например `finpulse_feedback_bot` (должен заканчиваться на `bot`)
5. Скопируйте **токен** — длинная строка вида `7123456789:AAH...`

**Токен никому не показывайте** и не заливайте в GitHub.

---

## Шаг 2. Узнайте свой chat_id (3 мин)

1. Найдите **своего нового бота** в Telegram
2. Нажмите **Start** / напишите `/start`
3. Откройте в браузере (подставьте свой токен):

```
https://api.telegram.org/botВАШ_ТОКЕН/getUpdates
```

4. Найдите `"chat":{"id":` — число, например `123456789`  
   Это ваш **CHAT_ID**.

---

## Шаг 3. Cloudflare Worker (бесплатно, 7 мин)

1. Регистрация: https://dash.cloudflare.com/
2. **Workers & Pages** → **Create** → **Create Worker**
3. Имя: `finpulse-feedback` → **Deploy**
4. **Edit code** → удалите всё → вставьте содержимое файла  
   `worker/telegram-feedback.js` из проекта → **Save and deploy**
5. **Settings** → **Variables** → **Add variable**:
   - `BOT_TOKEN` = токен из шага 1 (Encrypt)
   - `CHAT_ID` = число из шага 2 (Encrypt)
6. Скопируйте URL воркера, например:  
   `https://finpulse-feedback.ваш-ник.workers.dev`

Проверка: отправьте тестовую форму на сайте — сообщение должно прийти в Telegram.

---

## Шаг 4. Пропишите URL на сайте

На GitHub отредактируйте `js/config.js`:

```javascript
feedbackEndpoint: "https://finpulse-feedback.ваш-ник.workers.dev",
```

Commit → через 1–2 мин форма на https://finraz.ru/ заработает.

---

## Как это выглядит для пользователя

- Блок «Предложить калькулятор или идею»
- Поля: сообщение + контакт (необязательно)
- Кнопка «Отправить»
- **Ваш Telegram не виден**

---

## Безопасность

- Токен бота только в Cloudflare (зашифрован)
- На GitHub — только публичный URL воркера
- Есть honeypot от простых ботов
- Лимит длины сообщения: 2000 символов
