## Шаг 1. Яндекс.Метрика (~5 мин)

1. Открой https://metrika.yandex.ru → **Добавить счётчик**
2. Адрес сайта: `https://diabloalex666.github.io/Calculators/`
3. Включи: вебвизор, карта кликов, отказов
4. Скопируй **номер счётчика** (только цифры, например `98765432`)
5. На GitHub отредактируй `js/config.js`:

```javascript
yandexMetrikaId: "98765432",  // ← твой номер
```

6. Commit → подожди 1–2 мин → обнови сайт

Проверка: в Метрике → **Проверить счётчик** — должна появиться галочка.

## Шаг 2. Реклама РСЯ (~10 мин + 1–3 дня модерация)

1. https://partner.yandex.ru → **Рекламная сеть (РСЯ)**
2. **Добавить сайт** → URL: `https://diabloalex666.github.io/Calculators/`
3. Категория: «Финансы» или «Сервисы»
4. Дождись модерации (письмо на почту)

После одобрения:

5. **Реклама на сайтах** → **Создать блок** → тип **Адаптивный**
6. Скопируй **ID блока** (формат `R-A-1234567-1`)
7. В `js/config.js`:

```javascript
rsyaBlockId: "R-A-1234567-1",  // ← твой ID
```

8. Commit — реклама появится в блоках на страницах автоматически.

**Важно:** не кликай по своей рекламе — могут заблокировать.
