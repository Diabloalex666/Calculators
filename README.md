# FinPulse — пассивный доход как хобби

Репозиторий: [github.com/Diabloalex666/Calculators](https://github.com/Diabloalex666/Calculators)

Сайт (после загрузки файлов и включения Pages):

**https://diabloalex666.github.io/Calculators/**

## Что внутри

| Страница | Зачем |
|----------|--------|
| `salary.html` | Зарплата на руки (НДФЛ) |
| `vacation.html` | Отпускные |
| `compound.html` | Накопления со сложным процентом |
| `budget.html` | Бюджет на месяц (localStorage + печать) |

Люди ищут такие запросы в Яндексе каждый день. Сайт заточен под **органический трафик** + реклама/чаевые.

## Быстрый просмотр локально

Откройте `index.html` двойным кликом или:

```powershell
cd F:\AI\side-income
python -m http.server 8080
```

Откройте http://localhost:8080

## Публикация на GitHub Pages

**Пошагово через браузер:** [DEPLOY.md](./DEPLOY.md)

Git на ПК не нужен — загрузите папку через **Upload files** на GitHub.

Пошагово — в **[MONETIZATION.md](./MONETIZATION.md)** (30–60 минут настройки один раз).

Кратко:

1. Зарегистрировать домен (.ru от ~200 ₽/год) или использовать бесплатный `*.github.io`.
2. Выложить файлы (GitHub Pages / Cloudflare Pages / Vercel).
3. Подключить **Яндекс.Метрику** и **РСЯ** (реклама).
4. Вставить свою ссылку **Boosty** в `js/config.js` и блок «Поддержать проект».

## Ожидания (честно)

| Этап | Реалистично |
|------|-------------|
| 1–2 мес | 0–500 ₽ (пока нет трафика) |
| 3–6 мес | 1 000–8 000 ₽/мес при 500–3000 визитов |
| 12+ мес | 5 000–25 000 ₽/мес если SEO пошло |

Это **хобби-пассив**, не замена работе. Зато после настройки — правки раз в месяц или реже.

## Что править под себя

- `js/config.js` — Boosty, Метрика, РСЯ
- `robots.txt`, `sitemap.xml`, `index.html` — заменить `YOUR-DOMAIN.ru`
- Название FinPulse → своё (Find/Replace по папке)
